# Vivek's Brutalist Portfolio

> A minimalist, typography-led portfolio website for a full-stack web developer, featuring a clean editorial layout, oversized typography, and subtle motion.

🌍 **Live Preview:** [https://vivek-portfolio-website-brutalist.vercel.app/](https://vivek-portfolio-website-brutalist.vercel.app/)

## ⚡ Features

- **Brutalist / Minimalist Aesthetic**: Stripped-back design relying on strong typography, spacing, and a monochrome (black, white, gray) palette.
- **Custom Animations**: 
  - Smooth scroll-reveal effects using `IntersectionObserver`.
  - Engaging vertical text-flip hover animations (`FlipLink`) for navigation.
- **Responsive Layout**: Uses a max-width container of `1440px` with a 3-column editorial structure on desktop that gracefully collapses to a stacked layout on mobile.
- **Fast & Modern**: Built with the latest tools including React 19 and Vite for lightning-fast performance.

## 🛠️ Tech Stack

- **Frontend Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Language**: JavaScript / JSX

## 🚀 Getting Started

To run this project locally on your machine:

1. **Clone the repository**
   ```bash
   git clone https://github.com/vivek8817/Vivek-Portfolio-Website-Brutalist.git
   cd Vivek-Portfolio-Website-Brutalist
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173` to view the website.

## 📁 Project Structure

- `src/App.jsx`: Main application content and project sections.
- `src/components/layout/`: Contains the shared structural components (Header, Footer, Main layout).
- `src/components/animations/`: Custom reusable animation components (`Reveal.jsx`, `FlipLink.jsx`).
- `src/index.css`: Tailwind configuration, global font imports (Inter), and custom design tokens.
